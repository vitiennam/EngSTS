# How to build:

```
npm run build
```

# How to run server:

```
npm run server
```

# Features:
* Search English word and show result on 3 website: Oxford, Cambrigde, Google Translate (work on IOS)
* Flash Card
* Save your history seached word

# JavaScript runtime environment:
* Nodejs
